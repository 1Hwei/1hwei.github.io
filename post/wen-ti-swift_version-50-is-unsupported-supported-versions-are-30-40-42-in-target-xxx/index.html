问题：SWIFT_VERSION '5.0' is unsupported, supported versions are: 3.0, 4.0, 4.2. (in target 'XXX')

2019-08-20

##原因
这个项目是用之前的Swift版本做的，现在用最新版的Xcode打开之后，用的是最新版的Swift5版本进行编译，所以不兼容。

##解决方法：
点击主项目-BulidSetting-搜swift-Swift Language Version改成低版本
