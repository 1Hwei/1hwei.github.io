xcode：出现 Safe Area Layout Guide before ios 9.0 的错

2019-07-29

解决办法
选中控制器，右边面板的Builds for 选择 iOS 9.0 and Later

取消使用 Use Safe Area Layout Guides 选项，去针对不同的设备写代码适配。
